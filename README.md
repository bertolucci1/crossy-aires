# Crossy Aires

Un juego estilo Crossy Road ambientado en Buenos Aires, desarrollado con Three.js.

## Descripción

Cruza las autopistas y avenidas más icónicas de Buenos Aires esquivando autos, colectivos y trenes. Desbloquea nuevos mundos y skins para tus personajes.

## Controles

- **PC:** Flechas del teclado o clic en los botones en pantalla.
- **Móvil:** Botones táctiles en pantalla.

## Cómo jugar

1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
   - **Nota:** Para que los modelos 3D y texturas carguen correctamente, se recomienda usar un servidor local (como Live Server en VS Code) debido a las políticas CORS de los navegadores.
